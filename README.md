FizzBuzz implementation in Erlang.

To compile and run:

    $ erl
    1> compile("fizzbuzz").
    {ok,fizzbuzz}
    2> fizzbuzz:run().
    ...
   
<ctrl-C> twice to exit.

I've included [an explanation of the implementation](https://github.com/aeden/fizzbuzz-erlang/blob/master/explained.md) for those who are interested.
